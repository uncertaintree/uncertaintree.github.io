The simple linear regression model:
The observation unit is denoted using index \(i=1,\ldots,n\).
For response variable \(Y_i\) - a random variable -, and input variable \(x_i\),
the simple linear regression model is given as: \[ Y_i=\beta_0 +\beta_1\cdot x_i + \epsilon_i, \quad\epsilon_i\sim\text{Normal}\left(0,\sigma_{\epsilon}^2\right) \] For estimation of regression parameters \(\left(\beta_0,\beta_1,\sigma_{\epsilon}\right)^\top\)
we need response observations \(y_i\).
This is a call to brms::brm in order to estimate this model: 

m <- brm(y ~ x, 
         prior =  c(prior(student_t(3, 0, 2), class = b)),
         family = "gaussian",
         data = df)
