1 Use for statistical teaching

Among others, this data can be used to introduce generalized regression models (Bernoulli / Binomial regression for survival, Gamma / … for height and / or diameter) and hierarchical models (a.k.a. ‘mixed effects models’ among frequentists).

4 Data availability

4.1 Data on growth and survival

The repository (Link) that provides the download link to the data doesn’t give any statement on the terms of usage. Further, the “Data availability” section on the Article’s website states “Data is publicly available through the University of Copenhagen - Electronic Research Data Archive (ERDA).”. There is no further information given. So it should be okay to use the data for educational purpose, and moreover, the following argumentation for data from Table 1 also applies here.

4.2 Data fom the article’s Table 1

It is okay to re-use data represented in Table 1 (it should be even more for this solely educational purpose), since recording data is no original work of authorship (following block-quote is from the article ‘copyrightability_of_tables_charts_and_graphs.pdf’ (University of Michigan), which I found as it was linked to here.):

“Charts, graphs, and tables are not subject to copyright protection because they do not meet the first requirement for copyright protection, that is, they are not “original works of authorship,” under the definitions in the Act (17 U.S.C. § 102(a).). At first glance, this probably doesn’t make much sense; if a researcher runs a series of experiments and collects a data set, isn’t that original, and aren’t they the author of it? In a sense, yes, but in the sense that’s important for copyright, no. Facts and data aren’t considered original works of authorship because they are not “created” so much as they are “discovered.” For example, if a scientist takes temperature readings at various locations over a period of years, she isn’t “creating” the data, she’s recording the data. If she keeps a log describing how she feels every day, and how the sunrise looks at the testing station, that’s original, creative, authorship. Recording natural phenomena is not."
\(\vphantom{.}\) Unknown author(s), University of Michigan.

5 On the form of scaling used here

We center scale and covariates similar to Dias et al. (2021):

“In regression modeling it is common practice to empirically standardize the covariates by subtracting the empirical mean of the observed covariate values and diving the result by the empirical standard deviation. Here we standardize the covariates based on the available domain expertise which improves interpretability and generalizability of the resulting inferences as we now show.”

Greenland and Mansournia (2015) write similarly: “…, we strongly advise that quantitative covariates be scaled in units that are contextually meaningful, important, and generalizable (recognizable and transportable)”.

6 R Code

library("plyr")
## Data from Diatta et al. (2021), Table 1:
## (Direct link: https://link.springer.com/article/10.1007/s13595-021-01098-5/tables/1)
tab_1 <- data.frame(Provenance = c("Diamenar", "Djigueri", "Karofane", 
                                   "Sudan", "Kirane", "Kankoussa", 
                                   "Chad", "Daiba", "Aite", "Di", 
                                   "Bissiga", "Kidira", "Somo", 
                                   "Ngane", "Sodera"),
                    Rainfall_mm = c(288, 309, 366, 408, 380, 294, 349, 
                                    437, 449, 637, 696, 611, 627, 570, 
                                    762),
                    WAI0_mm = c(28, 45, 51, 56, 59, 60, 64, 65, 75, 
                                121, 128, 184, 187, 194, 292))
## Data from repository (alternatively using 'encoding = "latin1"'):
link <- "https://erda.ku.dk/archives/845762612be1bf53dd482052126ce220/Growth_A.%20senegal_AFS.csv"
df <- read.csv(link, stringsAsFactors = F)
rm(link)
names(df) <- gsub(names(df), pattern = "..", replacement = "_", 
                  fixed = T)
names(df) <- gsub(names(df), pattern = ".", replacement = "", 
                  fixed = T)
df$Provenance[df$Provenance == "A\xefte"] <- "Aite" ## As in Table 1
df$Provenance[df$Provenance == "Da\xefba"] <- "Daiba" ## As in Table 1
## There is no tree ID included in the data, which makes it impossible 
##   to get to realiable information on the longitudinal developments of 
##   single trees. 
## So only use the first measurement from the year 2008:
df <- subset(df, Year == 2008)
## Combination of location and provenance in a new variable:
df$Loc_Pr <- paste0(df$Location, "_", df$Provenance)
## Reliably organize survival and height and diameter:
##    Auxiliary variable 'line':
df$line <- 1:nrow(df)
## a) No available height but available survival (survived!) should be 
##    treated as 'dead' and therefore no height available:
(tmp <- subset(df, is.na(Height_m) & (Survival == 1)))
df$Survival[tmp$line] <- 0
df$Diameter_cm[tmp$line] <- NA
xtabs(~ is.na(Height_m) + Survival, data = df)
xtabs(~ is.na(Diameter_cm) + Survival, data = df)
## b) No available height but available survival -- but dead! -- should 
##    be treated as no available height and no available diameter:
(tmp <- subset(df, !is.na(Height_m) & (Survival == 0)))
df$Height_m[tmp$line] <- NA
df$Diameter_cm[tmp$line] <- NA
xtabs(~ is.na(Height_m) + Survival, data = df)
xtabs(~ is.na(Diameter_cm) + Survival, data = df)
## c) No available diameter but available survival (survived!) should 
##    be treated as dead and therefore also no height available:
(tmp <- subset(df, is.na(Diameter_cm) & (Survival == 1)))
df$Survival[tmp$line] <- 0
df$Height_m[tmp$line] <- NA
xtabs(~ is.na(Height_m) + Survival, data = df)
xtabs(~ is.na(Diameter_cm) + Survival, data = df)
## d) No available diameter and available survival -- but dead! -- should 
##    be treated as no available height and no available diameter:
(tmp <- subset(df, !is.na(Diameter_cm) & (Survival == 0)))
df$Height_m[tmp$line] <- NA
df$Diameter_cm[tmp$line] <- NA
xtabs(~ is.na(Height_m) + Survival, data = df)
xtabs(~ is.na(Diameter_cm) + Survival, data = df)
## Check distribution of height and diameter for survived trees:
summary(df$Diameter_cm) ## ... all greater 0.
summary(df$Height_m) ## ... all greater 0.
## Code height and diameter of non-survived trees with $0$:
df$Height_m[which(is.na(df$Height_m))] <- 0
df$Diameter_cm[which(is.na(df$Diameter_cm))] <- 0
## Integrate information on rain and water availability index (WAI0):
df <- merge(df, tab_1, by = "Provenance")
## Centering and scaling of x
summary(df)
f <- function(x){
  c(summary(x), 'sd' = sd(x), 'delta_range' = diff(range(x)))
}
f(df$WAI0_mm)
f(tab_1$WAI0_mm)
f(df$Rainfall_mm)
f(tab_1$Rainfall_mm)
df$WAI_c <- (df$WAI0_mm - 100) / 75
f(df$WAI_c)
df$Rain_c <- (df$Rainfall_mm - 500) / 150
f(df$Rain_c)
